Descripción

Mortero seco cementicio de alta calidad, formulado con áridos seleccionados, cemento y aditivos poliméricos. Diseñado para el revestimiento y regularización de paredes de bloque, arcilla y concreto, tanto en interiores como exteriores. Base perfecta para estuco y pintura.

Composición

Cemento Portland, áridos silíceos seleccionados, carbonatos de calcio, polímeros redispersables (RDP) y aditivos hidrófugos y plastificantes.

Ventajas

  • Excelente adherencia en bloque, arcilla y concreto
  • Fraguado controlado, fácil aplicación
  • Hidrófugo: resistente a filtraciones
  • Acabado liso listo para estuco o pintura
  • Proyectable manual o con máquina

Campo de aplicación

Revestimiento interior y exterior de paredes de bloque, arcilla y concreto. Primera capa de acabado antes del Estuco KORATEX® y Pintura Koracolor®. Base recomendada para todo el sistema de acabado KORATEX®.
